dbt Jumpstart





This project helps jump start any dbt project. You can simply collect all the information regarding your tables and their schema and upload here, and the service generates the dbt models for you. The following instructions will guide you through the process.

Preparation

You need to create two .xlsx spreadsheets - one that contains all the table configurations and another that contains the details about all the columns in the tables. The files must be named exactly as shown below:

  • tables.xlsx
  • columns.xlsx
